### How to Deploy Backend
## Requirements  
IIS 7  
.net framework 4.5  
turn on asp modules  

## Fontend  
1. Turn on IIS component  
2. Put the the backend folder to C:\inetpub\API  
3. Open IIS manager, Add new site  
4. Named Backend and select C:\inetpub\API as directory  
5. Enter port number 5001  
6. Start Site
